My Cybersecurity Projects

A hands-on showcase of my technical work, research, and tools.

Penetration Testing Suite

This project includes a suite of penetration testing tools developed using Python and integrated Bash scripts. The tool automates scanning, vulnerability identification, and basic exploitation techniques in a safe lab environment.

  • Custom Nmap-based network scanner
  • Automated directory brute-forcer using Python
  • Exploit chaining with Metasploit modules

Network Traffic Analyzer

This project analyzes live network traffic using Scapy and Wireshark. I developed scripts to detect potential anomalies like port scanning, malformed packets, and ARP spoofing attempts.

  • Live traffic capture with real-time alerts
  • Visualization with Matplotlib and Dash
  • Report generation via LaTeX and PDF export

Web Security CTF Challenges

I participated in Capture the Flag (CTF) challenges hosted by platforms like Hack The Box and TryHackMe, focusing on web exploitation and privilege escalation tasks.

  • SQL Injection, XSS, and CSRF exploitation

  • Documented writeups published on my cybersecurity blog
